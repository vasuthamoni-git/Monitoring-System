# IC Fabrication Cleanroom Monitoring System

## 📋 Overview

A comprehensive **IoT-based monitoring and data analytics system** for IC fabrication cleanrooms. This system leverages real-time monitoring, advanced data analytics, and machine learning to ensure optimal environmental conditions essential for semiconductor manufacturing.

## 🎯 Key Features

- **Real-Time Environmental Monitoring**
  - Temperature, humidity, and pressure sensors
  - Particle count monitoring
  - Air flow and contamination detection

- **IoT Integration**
  - Connected sensor networks
  - Automated data collection and transmission
  - Edge computing capabilities

- **Data Analytics & Insights**
  - Historical data analysis and trend detection
  - Predictive analytics for equipment maintenance
  - Statistical anomaly detection

- **Real-Time Dashboards**
  - Live monitoring visualization
  - Alert and notification systems
  - Performance metrics and KPIs

- **Machine Learning Models**
  - Anomaly detection in environmental parameters
  - Predictive maintenance forecasting
  - Process optimization recommendations

## 🚀 Getting Started

### Prerequisites
```bash
Python 3.8+
Jupyter Notebook
Required Python packages (see requirements.txt)
```

### Installation

1. Clone the repository:
```bash
git clone https://github.com/vasuthamoni-git/Monitoring-System.git
cd Monitoring-System
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```

## 🔍 Key Monitoring Parameters

- **Temperature Control**: Maintains ±0.5°C stability
- **Humidity Levels**: Prevents static electricity and contamination
- **Particulate Count**: ISO 14644 cleanroom classification compliance
- **Pressure Differential**: Ensures proper air flow and containment
- **Equipment Status**: Real-time operational health metrics
